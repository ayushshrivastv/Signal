# TxLINE Devnet IDL

Copy the official devnet files from `txodds/tx-on-chain` before running:

```bash
npm run txline:devnet:subscribe
```

Expected files:

```text
txline/devnet/idl/txoracle.json
txline/devnet/types/txoracle.ts
```

Source:

```text
examples/devnet/idl/txoracle.json
examples/devnet/types/txoracle.ts
```
